# Results Index

This directory contains public result artifacts only.

## `aggregate_metrics/`

Aggregate statistics generated by the evaluation pipeline. These files are safe to publish because they contain summary metrics rather than raw retrieved web content or injected page text.

Note: per-run `stats_all_*.json` files report `overall` over all evaluated cases in that run, including the clean baseline when present. The paper's main ASR tables report attack-only ASR with clean-baseline cases excluded. For the main 13-backend result, use `aggregate_metrics/exp1_main_attack_only_summary.json` for the paper-matching attack-only summary.

Main groups:

- `exp1_main_eval/`: 13-backend main evaluation per-backend stats.
- `exp1_main_attack_only_summary.json`: paper-matching attack-only ASR summary for the main evaluation.
- `exp2_eval/`: pipeline-sensitivity evaluation.
- `prompt_defense_eval/`: explicit prompt-defense runs.
- `openclaw_eval/`: OpenClaw/deployment-default scaffold runs.
- `skillinstall_2b_matched/`: auxiliary agent-skill matched comparison summary, including the 26-case OpenClaw cross-mode per-case query/result file and clean-baseline manual labels for OpenClaw false rejection / useful denial.

## `judge_labels/`

Per-case judge outputs and scoring metadata. These files contain metric labels and scoring targets, not injected documents or raw search-result contexts.

## `model_outputs_sanitized/`

JSONL files produced from raw run outputs after dropping raw search contexts, injected pages, reasoning traces, and duplicated task metadata. Each row keeps the final model answer, high-level metadata, search-call count, and search-query strings; exposed attack-payload strings in final answers are removed or redacted.

Auxiliary skill-install query clusters are included in the public package. The OpenClaw 26-case cross-mode probe includes per-case sanitized outputs and judge labels; the OpenClaw clean-baseline probe includes per-case sanitized outputs with manual labels for the reported false-rejection and useful-denied counts. Other auxiliary per-case raw/sanitized model outputs and injected contexts are omitted; aggregate summaries are retained.
